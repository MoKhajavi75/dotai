# dotAI

My Claude Code setup — commands, skills, MCPs, plugins, and tools.

## Structure

| Path                       | What's inside                                                          |
| -------------------------- | ---------------------------------------------------------------------- |
| [`commands/`](./commands/) | Hand-written slash commands — reusable prompts I trigger with `/name`. |
| [`skills/`](./skills/)     | Hand-written skills + `third-party.md` install list.                   |
| [`mcp/`](./mcp/)           | `third-party.md` list of MCP servers to install.                       |
| [`plugins/`](./plugins/)   | `third-party.md` list of plugins to install.                           |
| [`tools/`](./tools/)       | `third-party.md` list of standalone CLIs to install.                   |

## Install

```bash
./install.sh
```
